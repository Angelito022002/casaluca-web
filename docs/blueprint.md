# **App Name**: EduMetric Dashboard

## Core Features:

- Métricas de Progreso del Estudiante: Visualiza métricas clave como el progreso general, el promedio de calificaciones y las horas de estudio para un seguimiento rápido del rendimiento académico.
- Cursos Activos y Progreso: Muestra una lista interactiva de cursos matriculados con su porcentaje de finalización, facilitando el acceso a los materiales del curso.
- Gráficas de Rendimiento: Presenta visualizaciones semanales y mensuales del rendimiento académico a través de gráficas interactivas para identificar tendencias y áreas de mejora.
- Panel de Tareas y Logros: Una sección consolidada que muestra tareas pendientes, fechas límite próximas y logros o reconocimientos recientes del estudiante.
- Análisis de Tiempo de Estudio: Detalla la distribución del tiempo de estudio por asignatura o tipo de actividad, ofreciendo insights sobre hábitos de aprendizaje.
- Botones de Acción Rápida: Proporciona accesos directos personalizables para navegar a clases en vivo, entregar tareas o consultar calificaciones directamente.
- Herramienta de Recomendaciones Personalizadas de Estudio: Un tool de IA que analiza el historial de progreso y rendimiento del estudiante para sugerir recursos adicionales o ajustar el plan de estudio, optimizando el aprendizaje.

## Style Guidelines:

- Esquema de color oscuro basado en el diseño 'Emerald Gold Luxury Dining' para una estética sofisticada y de alta gama. El color de fondo principal es Dark Greenish Black (#0B1C16), creando un ambiente nocturno. El color de las superficies y tarjetas es Dark Emerald (#0F261E).
- El color primario es un oro cálido (#C5A059), utilizado para encabezados, elementos interactivos clave y detalles. Se utiliza un oro apagado (#8E7341) para acentos sutiles, bordes y elementos de menor énfasis, añadiendo profundidad al contraste.
- Los textos principales serán blancos (#FFFFFF), con un color secundario de texto gris claro (#BDBDBD) para descripciones y metadatos, siguiendo el esquema de color del sistema de diseño.
- Las tipografías recomendadas, según el sistema de diseño, son 'Playfair Display' (serif) para encabezados (weight 600) por su estilo elegante de alto contraste; 'Cormorant Garamond' (serif) para subtítulos (uppercase, weight 400) para un toque distintivo; y 'Lato' (sans-serif, weight 300) para el cuerpo del texto para una legibilidad limpia y moderna. Note: currently only Google Fonts are supported.
- Se emplearán iconos minimalistas y estilizados en color dorado, preferentemente delineados, para mantener la elegancia y coherencia visual con la estética de lujo del diseño, utilizados para las acciones rápidas y la indicación de métricas.
- El dashboard se estructura mediante un diseño de tarjetas (card-based layout), donde cada métrica o sección está contenida en un módulo de fondo Dark Emerald (#0F261E), tipografía prominente y acentos finos en dorado, sin bordes visibles para un aspecto limpio y premium. La disposición es centrada y simétrica, con abundante espacio negativo, utilizando una cuadrícula responsiva de 12 columnas.
- Se implementarán animaciones sutiles y elegantes para las interacciones del usuario, como 'gold glows' (brillos dorados) en estados de hover y clic para botones y elementos interactivos, y transiciones fluidas para actualizaciones de gráficas o expansión de contenido, acorde con la dirección visual de lujo sutil.
